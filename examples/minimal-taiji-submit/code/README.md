# Placeholder Code Directory

Replace this directory with your project code before building `code.zip`.

Keep the zip layout aligned with `run.sh`. For example, if `run.sh` executes:

```bash
python3 code/train.py --config config.yaml
```

then `code.zip` should contain `train.py` at its root, so extracting it into
`code/` creates `code/train.py`.
